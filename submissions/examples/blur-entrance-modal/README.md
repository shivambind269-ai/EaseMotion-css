# CSS Blur Entrance Modal

A responsive **Blur Entrance Modal** built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Blur + fade entrance animation
- Glassmorphism UI
- Fully responsive
- No JavaScript required
- Supports `prefers-reduced-motion`

## Folder Structure

```
blur-entrance-modal/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

The modal appears with a smooth blur and fade entrance animation.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Animations are automatically disabled for users who enable **Reduce Motion**.
