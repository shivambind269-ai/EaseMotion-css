# Bounce Pulse Feature Grid

A lightweight CSS-only feature grid with smooth bounce and pulse hover animations.

## Features

- Pure HTML and CSS
- No JavaScript required
- Responsive grid layout
- Bounce pulse hover interaction
- Accessible reduced motion support


## Usage

Add feature cards:

```html
<div class="feature-card">
    <div class="icon">⚡</div>
    <h2>Feature</h2>
    <p>Description</p>
</div>
