# Issue 18295 - Scroll Fade

Content blocks that fade in as they enter the viewport using IntersectionObserver.

## Usage

Open `demo.html` and scroll down. Each block fades in with a staggered delay.

## Files

- `demo.html` - Demo page with fade-in blocks
- `style.css` - Styles with fade animation and staggered delays
