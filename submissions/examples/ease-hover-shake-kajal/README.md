# ease-hover-shake

## Description

`ease-hover-shake` is a reusable CSS hover animation that applies a short horizontal shake effect to an element. It is built using pure HTML and CSS with no JavaScript.

## Features

- Short shake animation on hover
- Pure CSS implementation
- Lightweight and reusable
- Easy to customize

## Usage

```html
<button class="btn-danger ease-hover-shake">
    Delete Account
</button>
```

## Files

- `demo.html`
- `style.css`
- `README.md`

## Why EaseMotion CSS?

This component adds a simple hover shake animation that complements the existing Motion Library effects while remaining lightweight and easy to use.
