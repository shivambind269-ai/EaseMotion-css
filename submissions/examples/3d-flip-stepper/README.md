# 3D Flip Stepper

A pure CSS animated stepper component with 3D flip card transitions.

## Features

- Pure HTML and CSS
- No JavaScript required
- 3D flip animation
- Smooth hover interaction
- Responsive layout
- Reduced motion accessibility support

## Usage

```html
<div class="step">
    <div class="card">
        <div class="front">
            <h3>Step</h3>
        </div>

        <div class="back">
            <p>Description</p>
        </div>
    </div>
</div>
