# ▶️⏸️ Dual-State Button – Play/Pause Toggle with Morph

> A sleek, animated button that seamlessly morphs between play and pause states with a smooth, fluid transition, perfect for media controls and state toggles.

---

## 📖 Description

The **Dual-State Button** is an elegant UI component that toggles between play and pause states with a morphing animation. The icon smoothly transitions from a play triangle to a pause bar icon, with the button background subtly shifting and the label updating to reflect the current state. The morph effect uses `cubic-bezier` easing for a fluid, organic feel.

This component is perfect for media players, audio/video controls, and any interface where users need to toggle between two states. The smooth animation makes the state change feel natural and satisfying, providing clear visual feedback that enhances the user experience.
