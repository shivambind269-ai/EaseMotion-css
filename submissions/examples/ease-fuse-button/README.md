# 🔥 Fuse Button – Glow & Burn Effect

> A dramatic button where a fuse visibly burns with glowing sparks on hover, creating an explosive "ignite" experience.

---

## 📖 Description

The **Fuse Button** is an immersive UI component that simulates a burning fuse with realistic glow and spark effects. When hovered, the fuse burns from left to right with a vibrant orange glow, accompanied by pulsing sparks at the burn tip. The button itself responds with a subtle glow and animated icon, creating a cohesive "ignite" experience.

This component is perfect for:
- **Call-to-action buttons** that need a dramatic entrance
- **Destructive actions** (delete, remove, reset)
- **Launch/start buttons** in gaming or creative applications
- **Theme-appropriate interfaces** (fire, heat, energy-related)
- **Immersive experiences** where visual feedback is key

---

## 🎯 Features

- ✅ **Realistic fuse burn** – smooth, linear burn animation
- ✅ **Dynamic spark effect** – pulsating glow at the burn tip
- ✅ **Glowing button border** – animated gradient on hover
- ✅ **Icon animation** – pulsing fire icon with color transition
- ✅ **Dark theme optimized** – designed for dramatic dark backgrounds
- ✅ **Click feedback** – pulse animation on interaction
- ✅ **Accessible** – `:focus-visible` support
- ✅ **Fully responsive** – adapts to all screen sizes
- ✅ **Pure CSS animation** – GPU-accelerated for smooth performance

---

## 🚀 Quick Start

1. Place `index.html` and `style.css` in the same folder
2. Open `index.html` in your browser
3. Hover over the button to watch the fuse burn
4. Click for an extra pulse effect
