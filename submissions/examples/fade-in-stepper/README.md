# Fade-In Stepper

A lightweight pure CSS fade-in stepper component for accessible web layouts.

## Features

- Pure HTML and CSS
- Smooth fade entrance animation
- Responsive design
- No JavaScript dependency
- Supports prefers-reduced-motion accessibility

## Usage

Add the stepper structure:

```html
<div class="stepper">

<div class="step fade-step">
    <span>1</span>
    <h3>Title</h3>
    <p>Description</p>
</div>

</div>
