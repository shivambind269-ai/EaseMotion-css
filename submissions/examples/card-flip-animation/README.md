# Card Flip Animation

## Description

Card Flip Animation is a reusable CSS utility that creates a smooth 3D flip effect on hover. It is ideal for profile cards, product cards, portfolio items, and interactive UI components.

## Features

- Smooth 3D flip animation
- Pure HTML & CSS
- Lightweight and reusable
- Easy to integrate

## Usage

```html
<div class="flip-card">
  <div class="flip-card-inner">
    <div class="flip-card-front">
      Front Side
    </div>

    <div class="flip-card-back">
      Back Side
    </div>
  </div>
</div>
```
