# Card Hover Effects Collection

This is a self-contained example demonstrating 6 modern, pure CSS hover effects for cards, seamlessly integrated with **EaseMotion CSS** utility classes.

## Features
- 🔍 **Zoom & Overlay:** Image zooms in while overlay with content fades in.
- 🎲 **3D Tilt:** Cards tilt in 3D space on hover using CSS transforms.
- 🖼️ **Border Reveal:** Animated borders draw around the card on hover.
- ✨ **Glow Effect:** Animated gradient border with glowing shadow effect.
- 📊 **Split Reveal:** Top and bottom content panels slide in from opposite directions.
- 🎨 **Color Shift:** Background color smoothly transitions with icon rotation.
- ♿ **Fully Accessible:** Respects `prefers-reduced-motion` for users who prefer less animation.
- 📱 **Responsive:** Adapts to mobile screens with adjusted heights.
- 🚫 **Zero JavaScript:** Entirely built with HTML and CSS.

## How to Use
1. Ensure the EaseMotion CSS CDN is linked in your `<head>`.
2. Link the `style.css` file.
3. Choose a hover effect class (`.hover-card--zoom`, `.hover-card--tilt`, etc.).
4. Add the appropriate HTML structure for each effect.
5. Customize content, images, and colors as needed.
