# Futuristic AI Prompt Input Component

## What does this do?
A sleek, glassmorphic AI search/prompt bar featuring a continuous animated gradient shimmer, glowing drop-shadows on focus, and a pulsating microphone button.

## How is it used?
Add the component structure to your HTML:

    <div class="ai-prompt-container">
      <input type="text" class="ai-prompt-input ease-fade-in" placeholder="Ask anything..." />
      <button class="ai-prompt-btn ease-pulse ease-hover-grow">
        <svg><!-- Microphone or Send Icon --></svg>
      </button>
    </div>

## Why is it useful?
AI interfaces are incredibly popular right now. Building a premium, animated prompt input showcases the power of EaseMotion CSS by combining multiple utility classes into a single, cohesive, futuristic component that developers can easily copy and use in their own AI apps.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
