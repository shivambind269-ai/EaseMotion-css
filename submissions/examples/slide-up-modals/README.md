# CSS Slide-Up Modal

A responsive **Slide-Up Modal** built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Smooth slide-up entrance animation
- Glassmorphism-inspired UI
- Fully responsive
- No JavaScript required
- Supports `prefers-reduced-motion`

## Folder Structure

```
slide-up-modal/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in your browser.

The modal smoothly slides up into view using CSS keyframe animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Users who enable **Reduce Motion** automatically receive a non-animated version.
