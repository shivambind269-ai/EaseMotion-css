# Shimmer-Sweep Feature Grid

A lightweight pure CSS feature grid with a smooth shimmer sweep hover animation.

## Features

- Pure HTML and CSS
- Smooth shimmer sweep effect
- Responsive grid layout
- No JavaScript dependency
- Accessible reduced-motion support

## Usage

Add the feature cards inside:

```html
<div class="feature-grid">
    <div class="feature-card">
        <h2>Feature</h2>
        <p>Description</p>
    </div>
</div>
